triangulate
(triangulate points)
Build a triangle mesh over points.

Usage:

    (def points [(->Point Ax Ay) (->Point Bx By) ...])
    (def triangles (triangulate points))
    (first triangles) -> (->Triangle (->Point Ax Ay)
                                     (->Point Bx By)
                                     (->Point Cx Cy))
